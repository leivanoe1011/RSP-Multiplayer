Paper Scissor Rock Game

The game will require for a choice between P, R, or S.

The game emphasizes on the utilization of storing and retrieving data from Firebase.

Learn more

The application will take advantage of the Firebase methods and real time capabilities. Using firebase functions to retrieve data at will or when a user interacts with the application that submits data into the firebase server. The following lessons were learned
  • Event handler will run synchronously with other JS app functions
  • The lesson above forced us to run app functions within Firebase Event Handlers
  • We can chain firebase child objects with it's properties to create event handlers or only query specific data points
